# TODO List App

タスクを追加・完了・削除できる、シンプルなWebアプリです。
フロントエンドからバックエンドAPI、データ保存までを自分で実装しました。

## デモ画面

（※ここに後でスクリーンショットを追加するとさらに良い）

## 主な機能

- タスクの追加
- 完了 / 未完了の切り替え（チェックボックス）
- タスクの削除
- 入力したタスクはサーバー側に保存され、再読み込みしても残る

## 使用技術

| 区分 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（フレームワーク不使用） |
| バックエンド | Node.js / Express |
| データ保存 | JSONファイル |

## システム構成

ブラウザ（画面操作）
　↓ fetchで通信
Expressサーバー（APIで処理）
　↓ 読み書き
db.json（データ保存）

フロントエンドからの操作を、自作したREST API（GET / POST / PATCH / DELETE）が受け取り、
サーバー側でデータを更新する構成です。

## 動かし方

```bash
# 1. 依存パッケージをインストール
npm install

# 2. サーバーを起動
node server.js

# 3. ブラウザでアクセス
http://localhost:3000
```

## 工夫した点・学んだこと

文系出身で独学を続けてきた中で、「画面の裏側がどう動いているのか」を理解したく、
フロントエンドだけでなくバックエンドのAPIサーバーも自分で実装しました。

特に、画面の操作（追加・削除・完了）が、fetchを通じてサーバーへリクエストされ、
サーバーがデータを書き換えて返す——という一連の流れを自分で組むことで、
「画面とサーバーがどう連携してWebアプリが成立しているか」を実践的に理解できました。
